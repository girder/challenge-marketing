What's a Challenge?

A Data-Driven Challenge is a problem with well-defined inputs and outputs to which competitors can submit results and/or the code to produce results to be automatically scored against a set of known "true" results. The submission closest to the true results wins!

View details »

Why Challenges?

As the age of Big Data™ leads to a proliferation of statistical and machine-learning methods for solving problems, it becomes ever more important to run the latest and greatest algorithms on real-world data. When designed properly, challenges can show you how state-of-the-art methods work on the data you care about rather than with toy, synthetic, or sample data.
